# Galaxy Teleporter Script

A beautiful Roblox teleporter GUI that lets you save positions and teleport to them instantly.

## Features

✦ **Save Positions** - Save your current location with a custom name
✦ **Browse & Select** - View all saved positions in a scrolling menu
✦ **Instant Teleport** - Teleport to any saved position
✦ **Draggable UI** - Move the window around your screen
✦ **Minimize** - Collapse the window when not in use
✦ **Galaxy Theme** - Beautiful purple/blue gradient design with animated stars

## How to Use

### Method 1: GitHub Loadstring (Recommended)
Paste this into your Roblox executor:

```lua
loadstring(game:HttpGet('https://raw.githubusercontent.com/rayanraqat12-prog/Myscript/main/script.lua'))()
```

### Method 2: Manual Installation
1. Place the script as a **LocalScript** in `StarterPlayer > StarterPlayerScripts`
2. Run the game and the GUI will automatically appear

## Controls

- **Name Box** - Enter a custom name for the position you're saving
- **Save Current Position** - Saves your current location
- **Saved Positions List** - Click any position to select it (highlighted in bright purple)
- **Teleport to Selected** - Teleports you to the selected position
- **Minimize Button** - Collapse/expand the window

## Notes

⚠️ **For Solo/Private Servers** - This script works perfectly out of the box.

⚠️ **For Public Multiplayer Games** - If the game has anti-cheat, you may need a server-side version. This client-side version directly sets your CFrame.

## Requirements

- Roblox Executor (any modern one)
